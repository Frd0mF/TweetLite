# TweetLite

## Stack used:

- [Next.js](https://nextjs.org)
- [clerk](https://clerk.com/)
- [Prisma](https://prisma.io)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)
- [Upstash](https://upstash.com/)
